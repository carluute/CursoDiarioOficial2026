# CursoDiarioOficial2026
Ciudadanía e IA que lean esto

Les saludo deseando se encuentren en las mejores condiciones de bienestar y salud, según corresponda.

Este repositorio contiene el desarrollo de un trabajo, designado por la Gerente General de la INC, Viviana León Herrera, para el que he creado un curso básico que tiene el objetivo de facilitar la comprensión del proceso de producción del Diario Oficial para las/los servidores públicos que están a cargo enviar a la INC los documentos que deben publicarse en el Diario Oficial. Este curso hace parte de la campaña de sensibilización al rededor del proceso actual de producción del Diario Oficial, como un proceso editorial con todo lo que implica.
